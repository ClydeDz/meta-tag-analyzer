About

Meta Tag Analyzer is an Azure DevOps task that provides you with the ability to automatically run a check on your website and generate a report of the meta tag information. Benefits:

  1. Assurance: Ensure that your last release hasn't changed anything unexpectedly that will be impact your online presence
  2. Automatic and part of your value stream: Don't neglect the significance of meta tag content by checking this on a one-off basis. Include this as part of your pipeline and part of each release artifact.
  3. Comprehensive and easy to read: Simply download this report and email it to your marketing team (or any other non-technical team) to proof-read.

Meta Tag Analyzer Azure DevOps task will:

  1. Generate an Excel report after analysing the meta tags of all your pages. It will look something like this.
  2. Log the analysis to the console too, so you get the same information from within the pipeline itself.

How to get it

Get Meta Tag Analyzer from the Visual Studio Marketplace and install it in your Azure DevOps organisation. Once installed, you can add this task to your build or release pipeline.
